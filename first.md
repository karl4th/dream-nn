Аннотация (Abstract) 
Современные архитектуры нейронных сетей, такие как Трансформеры и модели пространства состояний (SSM, например, Mamba), достигли значительных успехов в обработке последовательностей, однако они остаются статичными после этапа обучения. В данной работе представлена архитектура NNAI-S, которая интегрирует принципы активного вывода (Active Inference) и синаптической пластичности (STDP) непосредственно в цикл инференса. 
NNAI-S использует механизм предсказательного кодирования для вычисления ошибки восприятия, которая модулирует "быстрые веса" через низкоранговое обновление. В архитектуру встроен механизм информационного гомеостаза (Surprise Gate) и цикл консолидации памяти (сон), что позволяет модели адаптироваться к нестационарным потокам данных (например, речи) в реальном времени, минимизируя вычислительные затраты и риск забывания. 
1. Введение (Introduction) 
Биологический мозг не является пассивным классификатором; это прогностическая машина, которая постоянно генерирует гипотезы о сенсорном входе. В отличие от современных ИИ, где веса фиксированы во время работы, биологические синапсы меняются ежесекундно. 
NNAI-S устраняет этот разрыв, объединяя эффективность SSM с динамикой активного вывода. Мы утверждаем, что такая архитектура является необходимым шагом к созданию автономных агентов, способных к непрерывному обучению без переучивания всей сети. 2. # Инициализация параметров 
U = torch.zeros(d_state, rank) 
U_target = torch.zeros(d_state, rank) 
V = torch.randn(d_model, rank).orthogonal_() # Фиксированный ортогональный фильтр eta = torch.randn(d_state, 1) # Векторная пластичность (broadcasting) W_target = 1.0 # Целевая норма для гомеостаза 
# Гиперпараметры сглаживания статистики 
beta = 0.05 # Коэффициент для mu и var 
beta_s = 0.1 # Коэффициент для накопленного удивления 
def step(x_t, h_prev, U_t, stats): 
# 1. Predictive Coding 
x_hat = C @ h_prev 
e_t = x_t - x_hat 
# 2. Surprise Gating 
H_t = 0.5 * torch.log(2 * torch.pi * torch.e * (stats.var + eps_sensory)) 
tau = tau_0 * (1 + alpha * H_t) 
S_t = torch.sigmoid((e_t.norm() - tau) / gamma) 
# 3. Fast Weight Update (STDP) 
# update: (d_state, rank) 
update = torch.outer(h_prev, e_t) @ V 
dU = -lambda_ * (U_t - U_target) + (eta * S_t) * update 
U_next = U_t + dU * dt
# 4. State Update (SSM / CfC) 
B_eff = (1 + kappa * S_t) * B_base + torch.tanh(U_next @ V.T) 
u_eff = B_eff @ x_t 
# ssm_transition: ODE-динамика (напр. CfC) 
h_next = ssm_transition(h_prev, u_eff) + W @ e_t 
# Обновление статистики (экспоненциальное сглаживание): 
# stats.mu = (1 - beta) * stats.mu + beta * e_t 
# stats.var = (1 - beta) * stats.var + beta * (e_t - stats.mu)**2 
# stats.avg_surprise = (1 - beta_s) * stats.avg_surprise + beta_s * S_t 
stats.update_surprise(S_t, e_t, beta, beta_s) 
return h_next, U_next, x_hat, S_t 
# Режим сна (вызывается отдельно после серии шагов) 
def sleep_consolidation(U_final, U_target, avg_surprise): 
if avg_surprise > S_min: 
dU_target = zeta_sleep * avg_surprise * (U_final - U_target) 
U_target += dU_target 
# Гомеостатическое масштабирование к норме W_target 
U_target = U_target * (W_target / (U_target.norm() + 1e-6)) 
return U_target 
# Основной цикл 
for batch in data: 
h, U = init_state() 
for x_t in batch: 
h, U, x_hat, S_t = step(x_t, h, U, stats) 
# Консолидация накопленного за батч опыта 
U_target = sleep_consolidation(U, U_target, stats.avg_surprise)

7.5 Ожидаемые результаты 
● Ожидается, что NNAI-S достигнет WER ≤ 8% на LJ Speech (сравнимо с CfC), но при смене диктора восстановит точность за < 50 шагов, тогда как статические модели потребуют полного переобучения. 
● Surprise Gate должен обеспечить стабильность CER при резком росте фонового шума. 
● Забывание: Ожидается, что после 10 циклов сна WER на старом дикторе упадет не более чем на 5% (против 30% у моделей без механизма консолидации).
